UVA RC Gen AI is available exclusively to users with active [Research Computing accounts](https://rc.virginia.edu/getting-started/get-access-high-performance-computing-resources/standard-security-zone-ssz#how-access-works). It's accessible via both a chat-like browser interface and API within standard security zone systems (Afton/Rivanna).  

## Where can I Access UVA RC GenAI from?
* **On grounds**: Direct browser access is available. API access must run from a standard security zone HPC node. See [HPC access methods](https://rc.virginia.edu/request-manage/ssz-login) for details.

* **Off grounds**: VPN is required for browser access and access to HPC nodes.

Additional details can be found in our [user guide](https://archive.rc.virginia.edu/userinfo/rcgenai-userguide/), and [support resources](https://forms.rc.virginia.edu/form/support-request/) can be found on our website.
