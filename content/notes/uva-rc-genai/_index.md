UVA RC GenAI is a UVA-hosted multimodal large language model (LLM) and vision-language model (VLM) server available to UVA research computing members with an active research computing account. Like other common multimodal platforms (e.g., ChatGPT or Claude), UVA RC GenAI can be used to assist with research, coding, and writing support, and all data is securely processed in-house at UVA. Currently, UVA RC GenAI is running the Kimi K2.5 model and is accessible via browser and API. UVA RC GenAI is only being offered for research use cases; it is not available for academic or administrative purposes.
