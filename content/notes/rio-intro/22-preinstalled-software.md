Every VM comes with a [base installation](https://www.rc.virginia.edu/userinfo/ivy/ivy-linux-sw/ "Software on Ivy Linux Virtual Machines page from the RC Website") of software by default. New software can be installed upon request. Software not already approved will need to undergo a security evaluation to determine if it is suitable for a High Security environment. 

Python and R packages are available to users through the normal `pip`, `conda`, and CRAN library installation methods.

Access to installed software is done using the lmod Module System.
